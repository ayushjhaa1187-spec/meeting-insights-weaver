# meeting-insights-weaver
AI-driven synthesizer for weaving disparate meeting notes into cohesive project milestones.

## Mission
Three meetings, four documents, and zero consensus. The project was drifting into chaos. He ran Weaver. One analysis. One result. Disparate notes snapped together into a single, undeniable roadmap. The team looked at the screen and finally saw the plan.

## Tech Stack
- Languages: JavaScript, Node.js
- - Tools: GPT-4, LlamaIndex
 
  - ## What This Does
  - Aggregates scattered meeting transcriptions and notes across multiple sessions to detect underlying consensus, action items, and project timelines.
